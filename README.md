# WiseLab Undergraduate Single-Cell RNA Sequencing Tutorial

Welcome to the WiseLab Undergraduate Single-Cell RNA Sequencing (scRNA-seq) Tutorial Repository.

This public repository hosts the files and datasets that are supporting materials used in the WiseLab tutorial series on processing and analyzing single-cell RNA sequencing data. 
The tutorial is designed to provide undergraduate students with hands-on experience in computational biology and bioinformatics workflows commonly used in modern genomics research.

## Learning Objectives

Through these tutorials, students will learn how to:

- Gain background knowledge on how single-cell sequencing is performed in R
- Perform quality control and filtering
- Normalize and scale gene expression data
- Identify highly variable genes
- Perform dimensionality reduction (PCA, UMAP)
- Cluster cells based on transcriptional profiles
- Identify marker genes
- Annotate cell populations
- Visualize and interpret single-cell datasets
- Compare and Contrast a disease state in scRNAseq data

## Repository Contents

- `healthyDonor_Test1234.combinedall.RDS` – Example datasets

- Seurat
- dplyr
- tidyverse
- HGNChelper
- openxlsx
- igraph
- ggraph
- data.tree

Additional package requirements are documented within individual tutorial files.

## Intended Audience

This repository was developed for undergraduate students participating in research training within the WiseLab at Salve Regina University and the RI-INBRE Workforce Development Training Program in Data Analysis. 
However, all materials are publicly available and may be used by anyone interested in learning the fundamentals of single-cell RNA sequencing analysis.
